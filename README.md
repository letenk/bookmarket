# Book Market
Book Market personal Project. Implementation Microservice with monorepo.

# Documentation

# Feature
* User Authentication (Login, Register, Forget password)
* User Profile
* Resource Book (Create, Read, Update, Delete)
* Order
* Payment gateway (Midtrans)

# Tech stack
* Golang
* PostgresSQL
* Docker
